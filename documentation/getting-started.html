Getting Started

Introduction

Thank you so much for purchasing the Edukidz Bootstrap html theme! Edukidz is a beautiful, modern, and one-of-a-kind theme that we developed from the ground up. It's jam-packed with cool features, and its main goal is to make your web pages shine out while remaining simple to use and maintain.

Downloading the theme

Use your usual zip software to unzip the zip file you downloaded from Template Monster.

When it's finished, you should be left with a folder containing the following files and directories.

Folder Contents
docs contains theme's documentation
root the theme folder and files

Installation

Follow these procedures to install the Edukidz Html theme via ftp.

  1. Use one of the various FTP applications, such as FileZilla, to connect to your website server.
  2. To install Edukidz, go to your default server folder and copy it there.

Setting your fonts

To use Google Fonts, go to Google Fonts and click on "Get a New Font." Google Fonts will offer you with a code to paste as the first element in the <head> of your HTML document once you've chosen the font styles you wish to load. This is how the code would appear, for example

<link href='https://fonts.googleapis.com/css2?family=Montserrat:wght@100;200;300;400;500;600;700;800;900&display=swap' rel='stylesheet' type='text/css'>

Add this rule to _variables.scss in the assets/scss/abstracts/ folder to allow the fonts to be integrated into your CSS.

$text-font-stack: 'Montserrat', sans-serif;

Keep in mind that various fonts can be used for different elements (as long as they are loaded in the <head> of your HTML document). For instance, if you want to change the font of your heading, you can use the CSS rule below.

h1, h2, h3, h4, h5, h6, .h1, .h2, .h3, .h4, .h5, .h6{
                            font-family: "Bubblegum Sans", cursive;
                            font-weight: 500;
                            font-size: 3.75rem;
                            color: #333333;
                        }

General Page Layout

The theme's pages use HTML5 semantic markup and are compatible with the most recent version of Twitter Bootstrap. The following elements make up the overall page design:

  1. Header: The website's banner and navigation are located in the header.
  2. Content: The page's main content.
  3. Footer: The page's footer is referred to as the footer.

The following markup is required for effective page markup.

<body data-spy="scroll" data-target="#mainNavbar" data-offset="91">
    <header class="main-navbar__wrapper">
                Menu Header
    </header>
    <div id="content" role="main">
        Main content
    </div>
    <footer class="site-footer home-page" id="footer" role="contentinfo">
        Footer
    <footer>
</body>
