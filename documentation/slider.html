Slider

Edukidz uses owl-carousel to generate dynamic slideshows that adapt to touch events. The following is the standard markup for making a slideshow.

<div class="owl-carousel owl-theme">
    <div class="item">
        A slide (any content)
    </div>
    <div class="item">
        Another  slide (any content)
    </div>
</div>

By passing the following settings to the slideshow, you can alter how it behaves:

  • autoplay: determines whether or not the slider will start automatically (true or false).
  • autoplayTimeout: The timeout for the autoplay interval (ex.7000).
  • loop: infinite loop. To create the illusion of a loop, duplicate the last and first objects (true or false).
  • nav: Display the next/previous buttons (true or false).
  • margin: margin-right(px) on item.
  • items: The number of items you want to see on the screen.
  • navText: HTML allowed.
  • dots: Show navigation with dots (true or false).
  • lazyLoad: Lazy load images. data-src and data-src-retina for highres. Also load images into background inline style if element is not
  • responsive: Object containing responsive options. Can be set to false to remove responsive capabilities.

Example of hero-slider options:

$('.js-main-slider').owlCarousel({
    loop: true,
    margin: 0,
    nav: true,
    autoplay: false,
    autoplayTimeout: 10000,
    items: 1,
    navText: ['', ''],
    dots: false,
    lazyLoad: true,
    responsive: {
        0: {
            items: 1,
            dots: true,
            nav: false,
        },
        1200: {
            items: 1,
            dots: false,
            nav: true,
        }
    }
});
Each slideshow should have an id or class.

Banner Slide Animation Animation of a Banner Slide This will be utilised to reveal the banner slide using animation. The following are the impacts and their corresponding classes:
  • Style 1: animation__style01
  • Style 2: animation__style02

Example of hero-slider added animation__style10:

<div class="js-main-slider owl-carousel owl-theme hero-owl">
    <div class="hero-slide animation__style10">
        A slide (any content)
    </div>
    <div class="hero-slide animation__style10">
        A slide (any content)
    </div>
</div>
