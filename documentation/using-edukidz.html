Page Content

The content <div id="content" role="main"> ...</div> section contains the theme's major content. To visually and conceptually separate the content, it is divided into sections.

One Page website

EduKidz is a single-page website for children. Two things must be ensured in order to do this. To begin, you must set a unique id to each of the sections that you want to scroll over, as seen in

<div id="home">
...
</div>

Second, you must ensure that each menu item connects to the appropriate part by using the # symbol followed by the section's id in the href attribute, as seen below.

<a class="nav-link main-navbar__nav-link js-scroll" href="#home">home</a>

As a result, clicking the Home menu option will scroll you to the top of the page to the home area. If we had a page with three sections, for example, it would look like this.

<body>
                            Add your menu here
                            <div id="home">
                                ...
                            </div>
                            <div id="aboutus">
                                ...
                            </div>
                            <div id="services">
                                ...
                            </div>
                        </body>

Now you have menu items that link to your sections, and when the user clicks on the appropriate link, the user will be taken to that area.
